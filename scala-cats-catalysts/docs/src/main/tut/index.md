## Installation

scala-cats-catalysts is available on scala 2.11, 2.12, 2.13 and scalajs.

Add the following to your build.sbt
```scala
libraryDependencies ++= Seq(
  "com.YourOrgName" %% "scala-cats-catalysts-core" % "0.0.1")
```
